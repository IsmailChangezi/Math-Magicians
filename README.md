# 📖 [Math-magicians]



> [Math Magicians] is a website for all fans of mathematics. It is a Single Page App (SPA) that allows users to: Make simple calculations. Read a random math-related quote

## 🛠 Built With <a name="built-with"></a>

- Frameworks: React.js
